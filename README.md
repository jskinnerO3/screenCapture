# ScreenCapture

A Windows screen capture and recording application built with WPF.

## Features

- **Screen Capture** - Capture entire screen or specific monitors
- **Window Capture** - Capture individual windows
- **Region Capture** - Select and capture custom regions
- **Scrolling Capture** - Capture scrollable content
- **Screen Recording** - Record screen with optional audio
- **Image Editor** - Annotate captures with:
  - Rectangles and ellipses
  - Arrows and lines
  - Text annotations
  - Highlighter
  - Blur tool
  - Crop tool
- **Global Hotkeys** - Quick capture with keyboard shortcuts

## Requirements

- Windows 10 or later
- .NET 8.0

## Building

```bash
dotnet build
```

## Running

```bash
dotnet run --project src/ScreenCapture.App
```
